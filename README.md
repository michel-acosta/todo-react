# To-Do App con React

## Descripción

Aplicación de gestión de tareas desarrollada con React. Permite añadir, completar, eliminar y filtrar tareas, con persistencia en localStorage.

## Tecnologías usadas

* React
* Vite
* Context API
* Custom Hooks
* Vitest + Testing Library

## Instalación

```bash
npm install
npm run dev
```

## Tests

```bash
npm run test
```

## Estructura

* components → UI
* hooks → lógica
* context → estado global
* tests → pruebas

## Funcionalidades

* Añadir tareas
* Marcar como completadas
* Eliminar tareas
* Filtrar tareas
* Persistencia en localStorage
* Lazy loading con React.lazy
* Tests unitarios
